## 1、下载项目  

```bash
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git 
```


创建 anaconda 环境, 建议 python 版本 3.10.6
```bash
conda create -n stable-diffusion python=3.10.6
```

进入环境
```bash
conda activate stable-diffusion
```

查看显卡信息
```bash
nvidia-smi
```

查看已经安装的 cuda 版本
```bash
nvcc --version
```
